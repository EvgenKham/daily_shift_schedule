# Страницы, URL и навигация (MVP)

В MVP приложение является SPA с маршрутизацией React Router и верхним меню.

## Карта маршрутов

- `/` → **Главная**
- `/roster` → **Наряд**
- `/schedule` → **График**
- `/settings` → **Настройки**
- `/help` → **Помощь**
- `/about` → **О нас**

### Поведение для неизвестных URL

- Любой неизвестный путь редиректится на `/`.

## Назначение страниц (MVP)

### Главная (`/`)

- Точка входа.
- Короткий статус/приветствие.
- Основной CTA: перейти в “Наряд”.

### Наряд (`/roster`)

- Выбор даты.
- Загрузка данных: настройки + график месяца.
- Генерация наряда по выбранной дате:
  - **Страница 1**: Выездные бригады (БИТ, Пед, Лин, 1170)
  - **Страница 2**: Вспомогательные службы (Диспетчерская, Заправочный блок, Уборщики)
- **Отображение в UI** (табличный вид, похожий на XLSX-пример):
  - Колонки: бригада день/ночь, состав, время прихода/ухода, подпись
  - Секции: выездные бригады, диспетчерская, заправочный блок, уборщики
  - Блок примечаний (опоздания, больничные)
  - Подписи (Врач СМП, Фельдшер)
- Редактирование наряда:
  - Inline-редактирование текста ячеек (ФИО, время)
  - Drag&drop swap (обмен сотрудниками между ячейками)
  - Редактирование примечаний
  - Undo последнего изменения
- Сохранение/восстановление (IndexedDB).
- Экспорт в XLSX (2 страницы, полное соответствие примеру).

### График (`/schedule`)

- Выбор месяца/года (ключ “YYYY-MM”).
- Импорт XLSX → JSON.
- Валидация и отображение ошибок/предупреждений.
- Сохранение графика месяца (IndexedDB).

### Настройки (`/settings`)

- Настройки пользователя и подстанции (ФИО, № подстанции, руководители).
- Конфигурация бригад (типы/номера/время старта и т.п. в рамках MVP).

### Помощь (`/help`)

- Статическая справка: как загрузить график, сгенерировать наряд, отредактировать, экспортировать.

### О нас (`/about`)

- Статическая страница “о приложении”.
