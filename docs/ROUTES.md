# Страницы, URL и навигация (MVP)

В MVP приложение является SPA с маршрутизацией React Router и верхним меню.

## Карта маршрутов

- `/` → **Главная**
- `/roster` → **Наряд**
- `/schedule` → **График**
- `/settings` → **Настройки**
- `/help` → **Помощь**
- `/about` → **О нас**

### Поведение для неизвестных URL

- Любой неизвестный путь редиректится на `/`.

## Назначение страниц (MVP)

### Главная (`/`)

- Точка входа.
- Короткий статус/приветствие.
- Основной CTA: перейти в “Наряд”.

### Наряд (`/roster`)

- Выбор даты.
- Загрузка данных: настройки + график месяца.
- Генерация наряда по выбранной дате.
- Редактирование наряда (inline + drag&drop swap + undo).
- Сохранение/восстановление (IndexedDB).
- Экспорт в XLSX.

### График (`/schedule`)

- Выбор месяца/года (ключ “YYYY-MM”).
- Импорт XLSX → JSON.
- Валидация и отображение ошибок/предупреждений.
- Сохранение графика месяца (IndexedDB).

### Настройки (`/settings`)

- Настройки пользователя и подстанции (ФИО, № подстанции, руководители).
- Конфигурация бригад (типы/номера/время старта и т.п. в рамках MVP).

### Помощь (`/help`)

- Статическая справка: как загрузить график, сгенерировать наряд, отредактировать, экспортировать.

### О нас (`/about`)

- Статическая страница “о приложении”.
