# Структура наряда (Daily Shift Roster)

На основе примера `пример наряда.pdf`.

## Общие сведения

- **Формат**: 2 страницы A4
- **Заголовок**: "Наряд" + дата (ДД.ММ.ГГГГ)
- **Ориентация**: книжная

---

## Страница 1: ВЫЕЗДНЫЕ БРИГАДЫ

### Таблица: 7 колонок

| № | Колонка | Описание | Пример |
|---|---------|----------|--------|
| 1 | Бригада\смена (день) | Тип бригады + номер + смена | БИТ 1110 8\20 |
| 2 | Состав бригады (день) | Список сотрудников (префикс "С") | С Рыжакова А.А.<br>С Сохлов В.А |
| 3 | Время прихода | Время начала смены | 8:00 |
| 4 | Подпись (день) | Пусто (для подписи) | |
| 5 | Бригада\смена (ночь) | Тип бригады + номер + смена | БИТ 1110 20\8 |
| 6 | Состав бригады (ночь) | Список сотрудников | С Рыжакова А.А.<br>С Н слабко Л.В. |
| 7 | Время прихода/ухода | Время | 20:00 |
| 8 | Подпись (ночь) | Пусто (для подписи) | |

### Типы выездных бригад

| Тип | Описание | Смены |
|-----|----------|-------|
| БИТ | Бригада интенсивной терапии | 8\20, 9\21, 20\8, 21\9 |
| Пед | Педиатрическая | 8\20, 20\8 |
| Лин | Линейная | 8\20, 9\21, 7\19, 20\8, 21\9, 19\7 |
| 1170 | Суточная бригада | 7.30\19.30 |

### Формат смены

- `8\20` — приход в 8:00, уход в 20:00 (дневная)
- `20\8` — приход в 20:00, уход в 8:00 (ночная)
- `9\21` — приход в 9:00, уход в 21:00
- `7\19` — приход в 7:00, уход в 19:00
- `7.30\19.30` — суточная с перерывом

### Блок "Примечания"

```
Примечания:

Опоздания, невыход на работу (больничный лист, повестка и т.д.)
[5 пустых строк для ввода]

Должность и Ф.И.О. лица, внесшего данные
[пустая строка]                              Роспись [пустая строка]

Врач СМП(Зав. п\с № 11)                      Юдаков Е.Ю.
Фельдшер(Старший) п\с № 11                   Лазарь Д.Г.
```

---

### Блок "Внимание!"

3 пункта инструкций:

1. Приступая к работе, включить радиостанцию и обеспечить ее на нахождение в автомобиле СМП.
2. При обнаружении очереди в приемном отделении какого-либо стационара, немедленно докладывать об этом старшему врачу оперативного отдела.
3. О любом изменении своего местоположения (приезд по адресу вызова, завершение вызова, освобождение в стационаре) или задержке на вызове более 1 часа обязательно сообщать диспетчеру.

## Требования к XLSX экспорту

### Стили

- Шрифт: Arial/Calibri, 10-12pt
- Границы: тонкие чёрные линии
- Выравнивание: по левому краю (текст), по центру (заголовки)
- Объединённые ячейки: заголовки секций
- Ширина колонок: авто или фиксированная

### Структура файла

- Лист 1: Страница 1 (Выездные бригады)
- Лист 2: Страница 2 (Вспомогательные службы)
- Или: оба на одном листе с разрывом страницы

## Данные для генерации наряда

### Источники

1. **График месяца** (import из XLSX)
   - ФИО сотрудников
   - Типы смен по датам
   - Номера бригад

2. **Настройки**
   - Подстанция №
   - Врачи/фельдшеры для подписей
   - Руководители

3. **Ручное редактирование**
   - Inline-редактирование ячеек
   - Drag&drop для обмена сотрудников
   - Примечания (опоздания, больничные)
